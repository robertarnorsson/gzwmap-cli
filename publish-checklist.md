# Publish Checklist
Use this checklist before publishing a new version.

- Update version in [index.js](./bin/index.js)
- Update version in [package.json](./package.json)
- Add new version in [README.md](./README.md)
- Run command ``npm i`` to update [package-lock.json](./package-lock.json)
- Run command ``npm publish`` to publish the new version
